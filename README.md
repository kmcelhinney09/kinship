# Kinship

A Family Command Center web application designed for Raspberry Pi and mobile devices.

## Features
- **Calendar**: Unified family view + individual calendars (Google Calendar Sync).
- **Meal Planning**: Recipe management, 2-week meal planner, grocery lists.
- **Chore Tracker**: Gamified system for parents and kids.

## Tech Stack
- **Frontend**: React, TypeScript, Vite, TanStack (Query, Router), TailwindCSS.
- **Backend**: FastAPI, SQLite, SQLAlchemy.
- **Deployment**: Docker (Raspberry Pi compatible).
